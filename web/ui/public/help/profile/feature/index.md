## 项目特性

```bash
- 丰富的配置
  - 是否启用登录/单点登录 （预留了流程 逻辑需各自实现）
  - 布局配置
  - 主题/主题色配置
  - 多环境发布配置

- 路由拦截、权限验证（启用单点登录模式后生效）

- 多环境发布（可按需扩展）
  - npm run dev 开发环境运行
  - npm run build:stage 阶段版本打包
  - npm run build:prod 生产版本打包

- 全局功能
  - SVGICON扩展
  - 本地/后端 mock 数据
  - 全局事件总线
  - 主题/主题色动态切换
  - 自定义控件（ucen-ui）

- 错误页面
  - 404

- 结合vscode eslint强制约束编码规范
```

>具体参见目录结构
